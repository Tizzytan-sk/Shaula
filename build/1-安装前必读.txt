Shaula Agent 安装前必读

适用范围：
- Windows 安装包：Shaula.Agent.Setup-<version>-x64.exe
- macOS 安装包：Shaula Agent-<version>-arm64.dmg

当前状态：
- Windows 安装包还没有代码签名，可能触发 Microsoft Defender SmartScreen。
- macOS DMG 还没有 Apple 开发者签名，可能触发 Gatekeeper quarantine。
- Shaula 只读取 Shaula 自己的数据目录，不会自动读取旧产品数据目录。

数据目录：
- Windows: %USERPROFILE%\.shaula
- macOS: ~/.shaula
- SDK 凭证目录仍沿用 ~/.pi，和上游 pi CLI 共享。


Windows 安装说明
================

1. 双击 Shaula.Agent.Setup-<version>-x64.exe。
2. 如果出现“Windows 已保护你的电脑”：
   - 先确认安装包来源可信。
   - 点击“更多信息”。
   - 点击“仍要运行”。
3. 按安装向导选择安装位置。
4. 安装完成后，从桌面快捷方式或开始菜单打开 Shaula Agent。

如果 Windows Defender 或公司安全软件拦截：
- 确认安装包来源后，将安装包或安装目录加入允许列表。
- 正式对外发布前应补 Windows 代码签名，减少 SmartScreen 拦截。


macOS 安装说明
==============

1. 打开 DMG。
2. 将 Shaula Agent 拖入 Applications。
3. 如果 macOS 提示：

   “Shaula Agent”已损坏，无法打开

   这通常不是安装包真的损坏，而是 macOS Gatekeeper 对未签名应用加了
   quarantine 标记。

4. 打开“终端”，执行：

   xattr -dr com.apple.quarantine "/Applications/Shaula Agent.app"

5. 然后再打开 Shaula Agent。

如果仍然打不开：
- 在“系统设置 > 隐私与安全性”里确认是否有允许打开 Shaula Agent 的提示。
- 或重新下载安装包后再执行上面的 xattr 命令。
